## Project Structure

- `index.html` — main portfolio page
- `style.css` — styling for layout, sections, cards, and responsive design
- `script.js` — optional interactive behavior (menu toggle, animations)
- `assets/images/` — local image assets used in the portfolio

## Image Assets

Ensure the following image files are placed in `assets/images/`:

- `portrait.jpg`
- `vendorbridge-preview.png`
- `ecommerce-preview.png`
- `vehicle-detection-preview.png`
- `sr-university.png`

## Local Preview

Open `index.html` in your browser or use a local server to preview the site.

## Deploy to Vercel

1. Push the repo to GitHub.
2. Connect the repository in Vercel.
3. Set the project root to `/` and deploy.

## Notes

- Remote SVG icons are loaded from CDN sources.
- All local images should use the `assets/images` path in `index.html`.
