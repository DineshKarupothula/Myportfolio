Vehicle Speed Detection & Number Plate Recognition

A real-time intelligent traffic monitoring system using Python, YOLOv5, OpenCV, and OCR technologies. The system performs vehicle detection, object tracking, speed estimation, and automatic license plate recognition from live video streams.

Built for automated traffic surveillance, the pipeline identifies overspeeding vehicles and extracts their license plate numbers through image preprocessing and OCR for real-time enforcement support.

Project Developer: Dinesh Karupothula

GitHub Repository
Detection Pipeline
  1. YOLOv5-based vehicle detection on live video frames.
  2. Object tracking across consecutive frames for continuity.
  3. Real-time speed estimation from frame displacement data.
  4. Overspeeding vehicle flagging with threshold alerts.
  5. OpenCV frame extraction and preprocessing pipeline.
Number Plate Recognition
  1. Automatic Number Plate Recognition (ANPR) from video streams.
  2. Image preprocessing for plate region enhancement.
  3. OCR-based character extraction from detected plates.
  4. Multi-format plate detection for varied conditions.
  5. Logging of detected plates with speed and timestamp.
Use Cases & Impact
  1. Automated traffic law enforcement and monitoring.
  2. Smart city surveillance for road safety improvement.
  3. Toll and checkpoint vehicle identification.
  4. Urban traffic flow analysis from video data.
  5. Accident hotspot monitoring and incident detection.

Technology Stack

  • Language: Python.
  • Detection Model: YOLOv5 (Ultralytics).
  • Video Processing: OpenCV.
  • OCR: Tesseract / EasyOCR.
  • Deep Learning: PyTorch.

System Architecture

  • Video input → frame extraction → YOLOv5 detection.
  • Object tracking across frames for speed calculation.
  • License plate crop → preprocessing → OCR extraction.
  • Flagging and logging of violations with full metadata.
